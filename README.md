# Disney Clone 
-A Disney+ Clone built using React.js and Firebase Firestore, featuring genre-based filtering, enhanced search results, and a trailer redirect system that opens YouTube trailers seamlessly. The application uses Firebase Authentication for secure user login and real-time data handling, and is deployed using Firebase Hosting to ensure fast and reliable performance.
## Features
-Genre-Based Filtering – Browse movies by categories such as Action, Kids, Family, Cartoon, Marvel, and more.

-Enhanced Search Results Page – Displays accurate and refined movie search results on a dedicated results screen.

-Trailer Redirect Feature – Instantly redirect to the movie’s YouTube trailer with a single click.

-Firebase Authentication – Secure user login and signup using Firebase Authentication.

-Firebase Firestore Integration – Real-time storage and retrieval of movie data and user information.

-Movie Slideshow Feature – A dynamic slideshow showcasing featured or trending movies.

-Firebase Hosting – Fast and reliable deployment using Firebase Hosting.
